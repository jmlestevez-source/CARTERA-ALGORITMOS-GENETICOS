# CARTERA ALGORITMOS GENETICOS 📈

Aplicación de gestión de cartera multi-sistema con datos de Yahoo Finance.

## Características

- 🔍 **Búsqueda de ETFs/Acciones** en Yahoo Finance con autocompletado
- 📊 **5 Sistemas de inversión**: CEG, DGSE, DDG, EIR, Colossus
- 📐 **Cálculo automático de asignación** con unidades enteras
- 💼 **Seguimiento de posiciones** con P&L en tiempo real
- 📈 **Gráficas interactivas** de evolución y distribución
- 💾 **Persistencia de datos** con export/import JSON
